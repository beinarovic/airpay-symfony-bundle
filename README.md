Airpay Management Bundle
=====================

A bundle for Symfony2.2, that manages AirPay payments.

1) Installation
---------------------

AirpayBundle can be installed via Composer.

<pre>
// composer.json
{
    // ...
    require: {
        // ..
        "beinarovic/airpay-symfony-bundle": "dev-master"

    }
}
</pre>
